# Surf Social Network Frontend

Фронтенд клиент социальной сети для сёрферов, разработанный на **Angular**. Приложение взаимодействует с REST API, реализованным на ASP.NET Core, и предоставляет интерфейс для регистрации пользователей, авторизации, просмотра и публикации постов.

Реализация [бэкенда](https://github.com/JaneeWatermelon/surf_site_server).

Проект разработан в рамках производственной практики в компании [СМС – информационные технологии](https://sms-it.ru/).

## Стек технологий

- Angular
- TypeScript
- HTML5
- CSS3
- RxJS
- JWT-аутентификация

## Основной функционал

### Пользователи

- Регистрация нового пользователя
- Авторизация
- Выход из аккаунта
- Автоматическая проверка авторизации при загрузке приложения
- Просмотр профиля пользователя

### Посты

- Просмотр ленты публикаций
- Создание новых постов
- Загрузка изображений
- Просмотр информации о публикации
- Отображение даты публикации
- Обновление списка публикаций после создания

### Валидация

- Проверка обязательных полей
- Проверка минимальной и максимальной длины
- Отображение ошибок, полученных с сервера
- Клиентская валидация с использованием Reactive Forms

## Требования

- Node.js 20+
- npm 10+
- Angular CLI

## Установка

Клонировать репозиторий:

```bash
git clone <repository_url>
```

Перейти в директорию проекта:

```bash
cd surf_site
```

Установить зависимости:

```bash
npm install
```

## Запуск

Запуск сервера разработки:

```bash
ng serve --open
```

После запуска приложение будет доступно по адресу:

```
http://localhost:4200
```

## Взаимодействие с API

Фронтенд работает с REST API.

Основные возможности API:

- регистрация пользователей;
- авторизация;
- получение информации о пользователе;
- создание публикаций;
- получение списка публикаций;
- загрузка изображений.

## Авторизация

Авторизация реализована с использованием **JWT**.
